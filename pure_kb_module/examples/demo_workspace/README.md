# Demo Workspace

这是纯知识库模块的最小测试目录。

当前包含一个最小 `demo` 领域测试说明，用于演示完整闭环。

1. 把 `pure_kb_module/local_kb/` 作为总知识库入口
2. 使用 `DEMO_QUICKTEST.md` 复测 `route / best-entry / digest / capture`
3. 如果效果 OK，再替换成新工程自己的真实领域知识

推荐顺序：

1. 先看根目录 `QUICKSTART.md`
2. 再执行 `DEMO_QUICKTEST.md`
3. 最后把 `demo` 替换成你的真实领域入口

入口：

- `DEMO_QUICKTEST.md`
